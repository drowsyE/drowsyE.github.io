시간 복잡도가 $O(N^2)$인 정렬 알고리즘들에 대해 알아보자.

## 1. 순차 정렬 / Sequential sort

가장 기본적인 정렬 알고리즘으로, 맨 앞에서부터 제일 작은 원소를 배치하게 만들어 나가는 알고리즘이다.

아래는 C로 구현한 순차 정렬이다.

```c
#define swap(a, b) {int temp = a; a = b; b = temp;} 

void sort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = i+1; j < n; j++) {
            if (arr[i] > arr[j]) {
                swap(arr[i], arr[j]);
            }
        }

        // for debug
        printf("Loop %d : ", i);
        for (int i = 0; i<n; i++) {
            printf("%d ", arr[i]);
        }
        puts("");
    }
}

int main() {
    int arr[] = {5,3,1,9,6,8,2,4,7};
    int n = sizeof(arr)/sizeof(arr[0]);
    sort(arr, n);
}
```
